Allow users on your BuddyPress site to enable or disable HTML-formatted emails created by the WP Better Emails plugin.

== Description ==

If you already use the awesome [WP Better Emails](http://wordpress.org/extend/plugins/wp-better-emails/) plugin on your [BuddyPress](http://wordpress.org/extend/plugins/buddypress) site, you'll notice that all your emails are sent out in HTML.

However, what if some of your users only want plain-text emails?

**Solution:** use this plugin! :)

Not publically released at the moment.

== Installation ==

1. Download, install and activate the [WP Better Emails](http://wordpress.org/extend/plugins/wp-better-emails/) plugin.
1. Download, install and activate this plugin.
1. Navigate to your BuddyPress notification settings page, you should see a new "Email Type" field at the very bottom.  If you want plain-text emails, select "No" and save.
